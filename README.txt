= gem-dump

https://github.com/dsisnero/gem-dump

== DESCRIPTION:

This package adds a gem dump command to dump your local gem list
into Gemfile format for easy install in new versions via
gem install --file gem.deps.rb

== FEATURES/PROBLEMS:



== SYNOPSIS:

gem dump --filename gem.deps.rb

== REQUIREMENTS:

requires rubygems 

== INSTALL:

gem install gem-dump

== DEVELOPERS:

After checking out the source, run:

  $ rake newb

This task will install any missing dependencies, run the tests/specs,
and generate the RDoc.
